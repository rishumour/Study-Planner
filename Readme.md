# Study Planner

A full-stack web application to help students plan, organize, and track their study sessions efficiently.

**Live Demo:** [https://study-planner-n3hd.onrender.com](https://study-planner-n3hd.onrender.com)  

## Features

- Add and manage subjects/courses
- Create study tasks with deadlines and priorities
- Visual progress tracking (e.g., completed vs pending tasks)
- Responsive UI for desktop and mobile
- Built with modern web technologies (e.g., React + Node/Flask – adjust to your actual stack)

## Local Setup

```bash
git clone https://github.com/rishumour/Study-Planner.git
cd Study-Planner

# Install dependencies
npm install         # or: pip install -r requirements.txt

# Configure environment variables (if any)
# e.g., copy .env.example to .env and update values

# Start development server
npm start           # or: python app.py / npm run dev
```

Then open `http://localhost:3000` (or your configured port) in the browser.

## Usage

1. Add your subjects (e.g., Data Structures, OS, DBMS).
2. Create study tasks with:
   - Subject
   - Task name
   - Priority (Low/Medium/High)
   - Due date
3. Mark tasks as done and track your progress visually.
4. Use the app regularly to stay consistent and avoid last-minute cramming.
